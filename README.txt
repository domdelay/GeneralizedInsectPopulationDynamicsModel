The general model can be run with the command: 
./execSingleCellRunner parametersFileName temperatureFileName outputFileName

with the command line arguments - 
parametersFileName: Valid filename and optional path to the parameters text file (.txt).
temperatureFileName: Valid filename and optional path to the temperature text file (.txt).
outputFileName: Valid output filename and optional path for the results text file (.txt).

- for more information see Delay 2022
